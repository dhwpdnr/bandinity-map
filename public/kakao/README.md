Kakao login button assets are downloaded from Kakao Developers resource endpoints.

- Source page: https://developers.kakao.com/tool/resource/login
- Source path pattern:
  - https://developers.kakao.com/tool/resource/static/img/button/login/full/ko/kakao_login_medium_wide.png
  - https://developers.kakao.com/tool/resource/static/img/button/login/simple/ko/kakao_login_medium.png

File naming convention in this folder:
- `kakao_login_medium_wide.png`: official full-width Korean button (used as primary asset)
- `kakao_login_simple_medium.png`: simple Korean button asset (kept for fallback/reference)
